# SingleValueForm

This is an example component.

```jsx
import { SingleValueForm } from 'ui-tydic';

export default () => <SingleValueForm list={[
    {
        label: 'name',
        value: 1,
    },
    {
        label: 'name1',
        value: 2,
    },
    {
        label: 'name2',
        value: 3,
    },
    {
        label: 'name3',
        value: 4,
    },
    {
        label: 'name4',
        value: 5,
    },
]} />
```
